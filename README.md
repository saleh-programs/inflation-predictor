# Inflation Predictor

This project uses data from various CPI sectors (gasoline, food, electricity, etc.) over the course of many years to produce ANNs and decision trees with reasonable accuracy!

##### The F1 scores (performance scores) for both models are provided:
- ANN model: 92%
- Decision Tree Model: 80%

It may be fed custom CPI data to determine if inflation is on the rise or not.
